动态规划：有递推关系，有状态转移方程

过程：递归 -> 记忆化搜索 -> 动态规划...

1. 状态定义，定义**子问题**f(i)

2. 初始状态，初始值

3. **状态转移方程**，转移方向（循环方向），填充值方向

4. 返回值

动态规划有两种计算顺序，一种是自顶向下的、使用备忘录的递归方法，一种是自底向上的、使用 dp 数组的循环方法。不过在普通的动态规划题目中，99% 的情况我们都不需要用到备忘录方法，所以我们最好坚持用自底向上的 dp 数组。

滚动数组等方法节省空间，缩减一个维度的空间
